# Concurrent-Matrix-Operations
A performance-optimized C++ project implementing concurrent matrix operations (transposition, zone summation, and self-multiplication). Designed to maximize execution speed on multi-core architectures without relying on pre-built parallel libraries or SIMD instructions, focusing on manual cache-blocking and thread pool synchronisation.
